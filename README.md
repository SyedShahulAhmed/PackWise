# PackWise 

PackWise is a trip management app built with **React.js ⚛️** and **Node.js 🌐**.  
You can create trips 🗺️, add items 📦, and toggle their **packed ✅ / unpacked ❌** status.

## Features 🚀
- Create and manage trips 🧳  
- Add items to trips 📝  
- Mark items as packed ✅ or unpacked ❌
